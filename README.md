# To-Do List Web App

## Objective
A simple front-end To-Do List application built using HTML, CSS, and Vanilla JavaScript.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Press Enter or click Add to add a task
- Instant UI updates without page refresh

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## Concepts Learned
- DOM Manipulation
- Event Listeners
- Dynamic UI Updates
- Class Toggle
- JavaScript Functions
